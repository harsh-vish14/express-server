## Run Locally

Clone the project

```bash
  git clone https://github.com/harsh-vish14/express-server.git
```

Go to the project directory

```bash
  cd express-server
```

Install dependencies

```bash
  npm install

  OR

  npm i
```

Start the server
having nodemon
```bash
  npm run start-mon
```
start the serve using node
```bash
  npm run start-node
```

server is running on http://localhost:8000/
